## Dependency
1) Node js
2) Truffle
3) Ganache
4) Metamask
ALWAYS RUN GANACHE FIRST BEFORE RUNNING THE APP

## Getting Started
```
npm install

*if npm install doesnt work and returns error ENOLOC, remove paclage.json file and run command again.*

npm install -g webpack webpack-cli

truffle migrate --reset
npm run start
```
